# 食譜收藏夾系統 (Recipe Collection System)

一個簡潔、直覺的個人食譜收藏與管理平台，讓你能輕鬆新增、搜尋、並透過清楚的步驟指引完成各種料理。

## 系統簡介

現代人生活忙碌，常常在網路上看到喜歡的食譜卻沒有一個統一的地方可以收藏與管理。市面上雖有許多食譜網站，但多數功能繁雜、廣告干擾多，缺乏針對「個人收藏與快速查找」的簡潔工具。
**食譜收藏夾系統**旨在提供一個乾淨的平台，讓使用者能輕鬆新增、搜尋、瀏覽食譜，並獲得個人化的推薦，同時清楚呈現每道料理所需的食材與步驟。

---

## 核心功能

- **輕鬆新增**：快速建立食譜（支援上傳圖片、所需食材與詳細步驟）。
- **多維搜尋**：依關鍵字、分類、難度與烹飪時間快速查找食譜。
- **個人收藏**：一鍵放入收藏夾，隨時調閱自己心儀的料理。
- **智慧推薦**：根據使用者偏好與歷史呈現可能感興趣的料理。

---

## 技術棧

- **前端：** HTML5 + CSS3 + 原生 JavaScript + Jinja2 模板
- **後端：** Python 3 + Flask 框架
- **資料庫：** SQLite + SQLAlchemy (ORM)
- **其他相依套件：** Flask-Login (認證)、Werkzeug (密碼安全)

## 本地開發指南

1. **建立並啟動虛擬環境**
   ```bash
   python -m venv venv
   source venv/bin/activate  # macOS / Linux
   ```

2. **安裝所需套件**
   ```bash
   pip install -r requirements.txt
   ```

3. **初始化資料庫**
   ```bash
   flask db init
   flask db migrate
   flask db upgrade
   ```

4. **啟動開發伺服器**
   ```bash
   python app.py
   # 或 flask run --debug
   ```
5. 開啟瀏覽器訪問 `http://127.0.0.1:5000`
